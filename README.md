# Godfather Ranker

**Breaking the North American Analytics Monopoly**

Open-source unified SEO analytics dashboard for EU SMEs. Unify 8+ data sources (Google Analytics, Search Console, Apify) into one sovereign, privacy-first dashboard.

## Why Godfather Ranker?

- **Market Problem:** Fragmented, expensive tools (€500+/mo) like Ahrefs/SEMrush create a "Digital Divide."
- **Our Solution:** A unified, AGPL-licensed alternative for **€10/mo** (self-hosted or community-hosted).
- **Target:** 3M+ EU SMEs currently priced out of professional SEO intelligence.

## Features

- ✅ **Unified Analytics:** Real-time data blending (Tag data + Rankings + Organic keywords).
- ✅ **Competitor Intelligence:** Automated tracking for 100+ competitors using Apify/Scraping logic.
- ✅ **Sovereign Hosting:** Fully Dockerized for easy deployment on EU-based servers.
- ✅ **FOSS Licensed:** Protected by **AGPL v3** to ensure the "Commons" stay open.

## Tech Stack

- **Frontend**: React 18 + TypeScript + D3.js (Advanced Data Viz)
- **Backend**: Node.js + Express + PostgreSQL (Time-series optimized)
- **APIs**: GA4, GSC, Apify SDK
- **DevOps**: Docker + GitHub Actions (CI/CD)
